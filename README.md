# 🧮 JavaScript Math Playground

An interactive learning platform to understand JavaScript's built-in `Math` object through live examples, explanations, quizzes, and hands-on experimentation.

---

## 📖 About

JavaScript Math Playground helps beginners understand how JavaScript Math functions work using real-time examples.

Instead of reading documentation, users can enter a number and instantly see the output of multiple Math methods with explanations.

---

## ✨ Features

- 🔢 Live Math Function Demonstrations
- 🎲 Random Number Generator
- 🌙 Dark / Light Mode
- 📚 Learn JavaScript Math Methods
- 🧠 Interactive Quiz
- 📱 Fully Responsive Design
- 🎨 Modern Card UI
- ⚡ Instant Results
- 💻 Beginner Friendly
